ellipsoid.projec_ellipse.generate_ellipsoid(a: float, b: float, c: float, num_points: int = 100) → tuple[ndarray, ndarray, ndarray][source]

Génère les coordonnées paramétriques d’un ellipsoïde.

Arguments :

a (float): Rayon le long de l’axe X. b (float): Rayon le long de l’axe Y. c (float): Rayon le long de l’axe Z. num_points (int): Nombre de points pour la grille paramétrique (résolution).

Retourne :

tuple[np.ndarray, np.ndarray, np.ndarray]: Coordonnées X, Y, Z de l’ellipsoïde.

ellipsoid.projec_ellipse.generate_orbit(center: tuple[float, float, float], radius: float, num_points: int = 100) → tuple[ndarray, ndarray, ndarray][source]

Génère une orbite circulaire autour d’un point donné.

Arguments :

center (tuple[float, float, float]): Coordonnées (x, y, z) du centre de l’orbite. radius (float): Rayon de l’orbite. num_points (int): Nombre de points pour la trajectoire orbitale.

Retourne :

tuple[np.ndarray, np.ndarray, np.ndarray]: Coordonnées X, Y, Z de l’orbite.
